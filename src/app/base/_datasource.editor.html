What's a Datasource?

A data source is a file or connection to a set of data Nomie Lab knows how to read.

Currently there are 3 types available:

  1. CouchDB interact with data in real time using Nomie's CouchDB support
  2. CSV Import a CSV that has been exported from Nomie
  3. Dropbox Import your data from a Nomie Dropbox Backup
